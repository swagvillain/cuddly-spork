# cuddly-spork
Dajngo app for education learning of important things 

## install
User will need to create a python virtual environment in the root directory of the project. User will need to source into it, then install (e.g. using pip) django and pytts3x. 

May need to run ```sudo apt install epseak-ng```

Then run the server, and navigate to 
```http://localhost:8000/myapp/```
